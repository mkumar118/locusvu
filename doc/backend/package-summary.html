Package backend

Class Summary

Database This class connects to the remote database via the mysql interface.
GetCytoBand This class contains methods to fetch cytoband data from the database.
GetGenes This class contains methods that submit sql queries to the database and fetch gene information for each locus.
GetOmim This class fetches OMIM data and creates an array (omimArray) to be later displayed in the results table.
GetRepeats This class contains methods to fetch Repeats information from the database.
GlobalParameters This class contains all the global variables.
Locus This is a generic Locus class, designed to hold and operate on the information specific to a locus.
OpenBrowser This class opens a link in the browser.
Pair A class of the generic type Pair, where a pair denotes a combination of start coordinate and end coordinate.
ReadInputFile Class to read the input file and generate the locusArray.
ResultsTable This class contains methods which handle display and organization of data in the results table.
Start This class contains the main method, the entry point for the tool.
Statistics This class contains methods that perform all calculations required to draw plots on the data.
TaskManager This class manages all the fetching of data from databases, and updating progress bar.
